Admin-related help

Creating a New Question

From the Home Page, click the 'Create New Question' button. This will open the Create New Question page.

You can also select the 'Create New Question' option from the navigation menu at the top-right of the page.

On the Create New Question page, select the type of question you would like to create. Once you provide a question type, additional options will be displayed below.

Enter the question's point value in the Point Value input. You must provide a whole number point value.

Select one or more categories from the Categories dropdown. You must provide at least one category for each question you create.

If your question does not fall under any of the available categories, you can create a new category by selecting the 'Create New...' option from the dropdown.

Additional options will be displayed after selecting a question type. These will vary based on the type of question you plan to create.

Enter the question text in the prompt below the Question Details header. This value is the question being asked of the student.

If you would like to provide an attachment for the question, select the 'Does this question have attachments?' checkbox. This will display an 'Upload File' button you can use to locate your attachment.

If you upload an image attachment, the image will be displayed within the question window.

For Multiple Choice and Checkbox questions, you will need to provide selectable options.

Within the 'Selectable Options' section, enter your options using the 'Add Option' button.

Select the 'Is this a correct answer?' checkbox for any correct answers. You must provide at least one correct option for each question.

For Checkbox questions, you may select more than one correct option. Students will need to select each correct option for their response to be considered correct.

For True or False questions, you will need to provide the correct response.

Provide the correct response using the True or False radio buttons.

For Short Answer questions, you will need to provide all acceptable responses.

If student responses will be case sensitive, select the 'Are acceptable answers case sensitive' checkbox.

Within the 'Acceptable Answers' section, enter any acceptable answers the student can provide.

For Upload questions, you will need to provide a correct answer file.

Upload the answer to the question using the 'Upload Correct Answer' button.

Once you have finished creating your question, select the 'Save Question' button to save your question.

 Creating a New Assessment

From the Home Page, click the 'Create New Assessment' button. This will open the Create New Assessment page.

You can also select the 'Create New Assessment' option from the navigation menu at the top-right of the page.

On the Create New Assessment Page, enter the name of the assessment you would like to create. The name will be displayed to students as they begin their assessment (e.g. "Starting BITS Assessment").

Provide a description of the assessment using the 'Assessment Description' text input.

Select the 'Assessment Configuration' tab to expand the contents.

If you would like the question order to be randomized, select the 'Randomize order of questions?' checkbox.

If you would like for the assessment to be timed, select the 'Is this a timed assessment?' checkbox. This will display a Time Limit input, where you can provide the time limit in minutes. Once the time limit is reached, the test will automatically end.

If you would like the assessment to end if the student provides multiple incorrect responses consecutively, provide the incorrect response limit in the 'Number of consecutive incorrect answers' input. Providing a limit of zero will not track consecutive incorrect answers.

To set the minimum passing grade for the assessment, use the slider to adjust the passing grade percentage.

Select the 'Add Questions to Assessment' tab to expand the contents.

Select the category of question using the 'Question Category' dropdown. This will display a list of each question that falls under that category.

You can click and drag to move questions from the 'Available Questions' list to the 'Assessment Questions' list.

The question order can be adjusted by dragging questions within the 'Assessment Questions' list. The question at the top of the list will be the first question provided to students.

If you do not want a set order for the questions, you can randomize the order using the 'Randomize order of questions?' checkbox found within the 'Assessment Configuration' tab.

When you are finished creating your assessment, use the save buttons found at the bottom of the page.

Selecting 'Save Assessment' will save the assessment and add it to the list of assessments that can be taken by students.

Selecting 'Finish Later' will save the assessment for later editing. Unfinished assessments are not available for students to take.

Advisor-related help

Generating an Assessment

From the Home Page, click the 'Generate Assessment' button. This will open the Generate Assessment page.

You can also select the 'Generate Assessment' option from the navigation menu at the top-right of the page.

Using the dropdown menu, select the assessment you wish to generate a URL for.

After you've selected an assessment, click the 'Generate Assessment URL' button.

The URL for the assessment will be copied to your computer. You can provide this URL to your student. Place this URL by pressing both the 'Ctrl' and 'V' keys on your keyboard.

You can also click the 'Start Assessment' button to open the assessment.

 Searching for a Student's Assessment Results

From the Home Page, click the 'View Student Results' button. This will open the Student Results page.

You can also select the 'View Student Results' option from the navigation menu at the top-right of the page.

Enter your search criteria within the Search Criteria section. You must provide one or more items to search.

To search for students based on your criteria, click the 'Get Results' button at the bottom-right of the section. This will display the results of your search.

Results are based on the criteria you provided. Results will match each piece of criteria you provided.

Under the Result Details column, click the icon to open the student's test results.

The student's test results are displayed here. Basic information is displayed at the top of the page.

Details on each question can be found below. You can see if the student's answer was correct or incorrect.

You can also view the question's point value, the question text, the student's submitted answer(s), and the correct answer(s).
